The API caches configuration tables in memory for performance. After editing values in the database (level configs, contact odds, distance weights, catch rates, etc.), you must clear caches for changes to take effect in simulations.


          

            Changed config and not seeing results? If you've modified level_distance_weights, level_contact_odds, level_catch_rates, level_batting_config, level_game_config, or other engine config tables, clear caches below before running your next simulation.

            Recalculates years and current_year for all active HS and College contracts
            so the frontend can derive class year correctly. Also moves 18-year-old college players back to HS.
          

          
    
            
  • HS contracts → 4-year, current_year = age − 14
    • 
            
  • College (non-RS) → 4-year, current_year = age − 18
    • 
            
  • College (redshirt) → 5-year, current_year = age − 18
    • 
            
  • Rebuilds contractDetails + contractTeamShare rows
