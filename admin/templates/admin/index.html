Analyze Player Data

          
Compute statistical breakdowns from current player data. This populates the config table with suggested mean/std values and quartile reference data. You can then edit the values below.


          

            When to analyze: After importing new players, completing a draft, processing trades, or anytime you want fresh baseline numbers. Levels with no players will be skipped.

Fix Amateur Contract Years

          

            Recalculates years and current_year for all active HS and College contracts
            so the frontend can derive class year correctly. Also moves 18-year-old college players back to HS.
          

          
    
            
  • HS contracts → 4-year, current_year = age − 14
    • 
            
  • College (non-RS) → 4-year, current_year = age − 18
    • 
            
  • College (redshirt) → 5-year, current_year = age − 18
    • 
            
  • Rebuilds contractDetails + contractTeamShare rows
